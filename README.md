# VUE-x-sunit-pwa

Front-end for testing X-SUNIT Back-end

# Configs

* Framework: **Vue JS**

# For using

### Install dependencies

```
    npm install
```

### Run serve for testing

```
    npm run dev
```

### Open in your browser

```
    http://localhost:9000/
```

__Obs: if you get error, please check if your port 9000 is free for runner the application__
